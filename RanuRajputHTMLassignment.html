What Is HTML?

  • HTML stands for Hyper Text Markup Language
  • HTML is the standard markup language for creating Web pages.
  • HTML describes the structure of Web page
  • HTML consist of series of element
  • HTML elements tells the browser how to display the content
  • HTML elements lable pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc.

What is an HTML Element?

An HTML is defined by a start tag, some content, and an end tag:

  <tagname>Content goes here...</tagname>

The HTML element is everywhere from the start tag to the end tag:

  <h1>My First Heading</h1>

  <p>My first paragraph.</p>
